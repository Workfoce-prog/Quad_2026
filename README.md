# quad_2026 (Regenerated Demo)

This bundle includes:
- `app.py` Streamlit app (MN Income Shares + GAI* demo)
- `data/mn_basic_support_table_template.csv` (structured statute table)
- `data/fpl_table_template.csv` (template values; replace with official FPL table if desired)

## Run locally
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deploy to Streamlit Cloud
Make sure the `data/` folder is committed to GitHub.
